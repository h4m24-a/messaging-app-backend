# Messaging App Backend

## View Project : 


**Tech Stack:** Node.js, Express, PostgreSQL, Prisma ORM, Passport.js, JWT, bcryptjs, AWS S3, Multer, Express Validator

- RESTful API for authentication, users, profiles, and messages
- Secure authentication and authorization with JWT
- Password hashing using bcrypt
- Database storage for users, profiles, and messages with PostgreSQL
- PostgreSQL database with Prisma ORM
- Persistent conversation history
- User lookup and conversation management
- Server-side validation with Express Validator
- Profile image uploads to AWS S3 using Multer
